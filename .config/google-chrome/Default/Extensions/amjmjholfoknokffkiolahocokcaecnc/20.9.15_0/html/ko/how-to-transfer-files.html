파일 공유하기

파일을 선택하면 링크가 생성됩니다.
링크만 있으면 어디서든 파일을 다운로드할 수 있어요.

* 링크의 사용기한은 7일이며, 기한 내에 횟수 제한 없이 다운로드할 수 있습니다.

링크 공유하기

이메일을 클릭하면 메일로 링크를 전달할 수 있습니다.
링크만 복사할 수도 있어요.

QR 코드

QR 코드로 생성된 링크를 쉽게 입력할 수 있습니다.
센드애니웨어 앱의 '받기' 메뉴에서 QR 코드를 스캔해보세요.

이 외에 센드애니웨어에서 가능한 일!

Gmail에 대용량 파일 첨부하기

대용량 파일을 메일로 보낼 때 센드애니웨어로 첨부하세요. 메일 작성 창의 센드애니웨어 아이콘을 누르면 첨부할 파일을 선택할 수 있습니다. 이미 업로드한 파일을 첨부할 수도 있어요.

PDF 공유하기

크롬 브라우저상의 PDF도 쉽게 공유할 수 있어요. PDF 뷰어의 Send Anywhere로 PDF 공유 버튼을 누르면 공유 링크가 생성됩니다.

Slack으로 대용량 파일 보내기

Slack에서 팀원들에게 대용량 파일을 공유할 때 센드애니웨어를 이용해 보세요. 메시지 입력창 옆 add 를 누르면 센드애니웨어로 파일을 첨부할 수 있습니다.

* 크롬 브라우저로 Slack에 접속했을 때만 사용 가능합니다.

사용 중 문의사항이나 불편한 점이 있으시면 고객센터로 문의해주세요
